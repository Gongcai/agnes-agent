I'd be happy to help you understand the design system! The Claude Desktop app uses a carefully crafted design language built around warm, paper-like tones and a signature terracotta accent color.

The color system uses HSL values stored as CSS custom properties, which enables smooth opacity variations. Here's an example of how tokens are defined:

:root {
  --bg-100: 48 33.3% 97.1%;  /* warm off-white */
  --brand-200: 15 63.1% 59.6%; /* terracotta #D97757 */
  --text-000: 60 2.6% 7.6%;   /* near-black warm */
}

The typography uses Anthropic's custom variable fonts — Anthropic Sans for UI and Anthropic Serif for display text, both supporting weights from 300 to 800.

That's great! Can you explain how the dark mode works?
Claude

Dark mode is triggered by prefers-color-scheme media query and applies a .darkTheme class to the <body> element. The background shifts from warm off-white (#faf9f5) to warm dark gray (#262624), while text inverts to warm whites.

Try the Light/Dark toggle in the top-right corner of this page to see the transition in action!
